    • Building better supply chains with blockchain technology and a new governance system.

    E ximchain is a Singapore-based company building blockchain applications for the supply chain space.

    A native wallet and governance system were needed to launch the main-net. Supply chains are global by nature - to ensure the legal compliance and integrity of the network, a comprehensive identity validation process was implemented.

    Iterating and extensively user testing several prototypes, we arrived at a simple mobile app to securely create a wallet, validate user identities, and view network members. A complementary desktop app was created to send and sign transactions, view token balances, and interact with smart contracts. By balancing ease of use with security through the native mobile app and desktop app combination, we increased our member network base and provided a trusted system to them. Using an agile sprint structure allowed us to launch the mobile app with key features on a tight timeline.

    Desktop Cryptocurrency Wallet

    Wallet App + Website

    I was responsible for the design and front-end development of the Eximchain Wallet app, an Electron-based desktop app powered by React. Forked from the popular MyCrypto crytocurrency wallet, our app featured a number of improvements, including a streamlined interface, a simpler transaction executor, and the ability to vote for which members of the Eximchain Network have control of masternodes. I also created the landing page for the Wallet marketing site.

    Token Holder Verification

    I created the process by which users verified that they held tokens. The Eximchain Network requires members to hold a minimum of 500 EXC and pass Know Your Customer (KYC) Verification to participate in voting cycles.

    Design System

    Axiom & Meta

    We named our design system Axiom because it is the source of truth for all of our designs. Utilising atomic design methodology allowed the system to be versatile and easy to update.

    Axiom is comprised of 4 sections: foundations, components, patterns, and templates. Documentation is directly posted in Figma files alongside UI elements.

    Under the hood of Axiom was the Meta Design System: a design system for the design system. This includes components like color swatch formats, typography specs, covers for design files, and formatting of documentation.
